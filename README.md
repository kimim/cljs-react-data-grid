# cljs-react-data-grid

demo of react-data-grid in clojurescript.

## Prerequisites

You will need [Leiningen][1] 2.0 or above installed.

[1]: https://github.com/technomancy/leiningen

## Running

To compile cljs:

    lein shadow compile :app

To start the server:

    lein run

Visit: http://localhost:3030
